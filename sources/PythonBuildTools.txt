1. Установить сборщик: pip install pyinstaller
2. Выполнить сборку, перейти в католог с файлом mail и выполнить: pip install pyinstaller

После завершения процесса компиляции, в каталоге dist появится исполняемый файл .exe.
Вы можете скопировать его в любое место и запускать программу без установки Python.
